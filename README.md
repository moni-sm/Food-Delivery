# 🍔 Premium 3D MERN Food Delivery App

A state-of-the-art, full-stack Food Delivery application featuring a **Premium 3D Minimalist** design, seamless navigation, and a robust MERN backend.

## ✨ Key Features

### 🚀 **Immersive 3D Experience**
- **Premium Minimalist 3D UI**: A refined, spatial interface using advanced CSS 3D transforms.
- **Floating 3D Dynamics**: Experience depth with images that physically float above cards and elements that react to interaction.
- **Pro Entrance Animations**: Every section slides in with sophisticated 3D reveals for a high-end "Pro" feel.

### 🍱 **Advanced Menu & Search**
- **Smart Real-time Search**: A functional search bar that filters the menu across the entire app.
- **Intelligent Navigation**: Auto-scrolls to results when categories are selected, ensuring a frictionless user journey.
- **Category Filtering**: Smoothly browse through categories with a refined, stable interaction model.

### 🛡️ **Robust Full-Stack Core**
- **MERN Architecture**: Built with MongoDB, Express.js, React, and Node.js for maximum performance.
- **Secure Authentication**: JWT-based user login and registration system.
- **Payment Integration**: Fully functional Stripe API integration for secure food ordering.
- **Admin Dashboard**: Comprehensive control panel for managing menu items and orders.

---

## 🛠 Tech Stack

- **Frontend**: React.js, Advanced CSS3 (3D Transforms, Glassmorphism)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **Security**: JWT (JSON Web Tokens)
- **Payments**: Stripe API
- **Context API**: Global state management for carts and search.

## 🚀 Quick Setup

### 1. Backend Configuration
- Navigate to `backend/`
- Run `npm install`
- Create a `.env` file with:
  ```env
  MONGO_DB="your_mongodb_uri"
  JWT_SECRET="your_secret_key"
  STRIPE_SECRET_KEY="your_stripe_key"
  ```
- Run `node server.js`

### 2. Frontend Configuration
- Navigate to `frontend/`
- Run `npm install`
- Run `npm run dev`
